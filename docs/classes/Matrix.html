Matrix Class

Extends Class
Defined in: src\math\Matrix.js:1
Module: go2d

变化矩阵类，实现矩阵基本运算，可表达两个二维向量空间之间的仿射变换。


            |a  b  0|
(x, y, 1) * |c  d  0| = (ax + cy + tx, bx + dy + ty, 1)
            |tx ty 1|
